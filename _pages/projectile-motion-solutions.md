# ✅ Projectile Motion — Solutions
{: .section-heading }

Full step-by-step solutions to all 20 problems from the [Exercises page]({{ '/projectile-motion-exercises/' | relative_url }}). All calculations use \\(g = 9.81 \, \text{m/s}^2\\).
{: .theory-section }

## ⭐ Level 1 — Warm-up
{: .section-heading }

<div class="solution-item" markdown="1">
<span class="sol-num">**1.**</span> \\(v_0 = 20 \, \text{m/s}\\), \\(\theta = 30°\\).

\\[v_{0x} = v_0 \cos\theta = 20 \times \cos 30° = 20 \times 0.866 = 17.32 \, \text{m/s}\\]
\\[v_{0y} = v_0 \sin\theta = 20 \times \sin 30° = 20 \times 0.5 = 10.00 \, \text{m/s}\\]

<span class="final-answer">v₀ₓ ≈ 17.32 m/s, v₀ᵧ = 10.00 m/s</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**2.**</span> \\(v_0 = 30 \, \text{m/s}\\), \\(\theta = 45°\\).

\\[T = \frac{2 v_0 \sin\theta}{g} = \frac{2 \times 30 \times \sin 45°}{9.81} = \frac{42.43}{9.81}\\]

<span class="final-answer">T ≈ 4.33 s</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**3.**</span> \\(v_0 = 15 \, \text{m/s}\\), \\(\theta = 60°\\).

\\[H = \frac{v_0^2 \sin^2\theta}{2g} = \frac{15^2 \times \sin^2 60°}{2 \times 9.81} = \frac{225 \times 0.75}{19.62}\\]

<span class="final-answer">H ≈ 8.60 m</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**4.**</span> \\(v_0 = 25 \, \text{m/s}\\), \\(\theta = 40°\\).

\\[R = \frac{v_0^2 \sin(2\theta)}{g} = \frac{25^2 \times \sin 80°}{9.81} = \frac{625 \times 0.9848}{9.81}\\]

<span class="final-answer">R ≈ 62.74 m</span>
</div>

## ⭐⭐ Level 2 — Getting Comfortable
{: .section-heading }

<div class="solution-item" markdown="1">
<span class="sol-num">**5.**</span> \\(v_0 = 22 \, \text{m/s}\\), \\(\theta = 35°\\), \\(t = 1.5 \, \text{s}\\).

\\[v_{0x} = 22 \cos 35° = 18.02 \, \text{m/s}, \quad v_{0y} = 22 \sin 35° = 12.62 \, \text{m/s}\\]
\\[x(1.5) = v_{0x} \cdot t = 18.02 \times 1.5 = 27.03 \, \text{m}\\]
\\[y(1.5) = v_{0y} t - \tfrac{1}{2} g t^2 = 12.62 \times 1.5 - 4.905 \times 1.5^2 = 18.93 - 11.04\\]

<span class="final-answer">x ≈ 27.03 m, y ≈ 7.89 m</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**6.**</span> \\(\theta = 50°\\), \\(R = 18 \, \text{m}\\).

\\[R = \frac{v_0^2 \sin(2\theta)}{g} \implies v_0 = \sqrt{\frac{R g}{\sin(2\theta)}} = \sqrt{\frac{18 \times 9.81}{\sin 100°}} = \sqrt{\frac{176.58}{0.9848}} = \sqrt{179.3}\\]

<span class="final-answer">v₀ ≈ 13.39 m/s</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**7.**</span> \\(v_0 = 18 \, \text{m/s}\\), \\(\theta = 42°\\), \\(t = 1.0 \, \text{s}\\).

\\[v_x = v_{0x} = 18 \cos 42° = 13.38 \, \text{m/s} \quad \text{(constant)}\\]
\\[v_y(1) = v_{0y} - g t = 18 \sin 42° - 9.81 \times 1 = 12.04 - 9.81 = 2.23 \, \text{m/s}\\]
\\[|v| = \sqrt{v_x^2 + v_y^2} = \sqrt{13.38^2 + 2.23^2} = \sqrt{183.9}\\]
\\[\phi = \arctan\left(\frac{v_y}{v_x}\right) = \arctan\left(\frac{2.23}{13.38}\right)\\]

<span class="final-answer">|v| ≈ 13.56 m/s, directed ≈ 9.5° above horizontal</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**8.**</span> \\(v_0 = 9 \, \text{m/s}\\), \\(\theta = 55°\\).

\\[t_{up} = \frac{v_0 \sin\theta}{g} = \frac{9 \times \sin 55°}{9.81} = \frac{7.37}{9.81}\\]
\\[H = \frac{v_0^2 \sin^2\theta}{2g} = \frac{81 \times 0.671}{19.62}\\]

<span class="final-answer">t_up ≈ 0.75 s, H ≈ 2.77 m</span>
</div>

## ⭐⭐⭐ Level 3 — Building Skills
{: .section-heading }

<div class="solution-item" markdown="1">
<span class="sol-num">**9.**</span> \\(v_0 = 28 \, \text{m/s}\\), \\(R = 40 \, \text{m}\\).

\\[\sin(2\theta) = \frac{R g}{v_0^2} = \frac{40 \times 9.81}{784} = \frac{392.4}{784} = 0.5005\\]
\\[2\theta = \arcsin(0.5005) = 30.06° \quad \text{or} \quad 2\theta = 180° - 30.06° = 149.94°\\]

Since sine is symmetric about 90°, every range (below the maximum) is reached by **two complementary angles**, \\(\theta\\) and \\(90° - \theta\\).

<span class="final-answer">θ ≈ 15.0° or θ ≈ 75.0°</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**10.**</span> Horizontal launch, \\(h = 20 \, \text{m}\\), \\(v_0 = 12 \, \text{m/s}\\).

With \\(\theta = 0°\\), all the initial speed is horizontal, so the fall time only depends on gravity:

\\[h = \tfrac{1}{2} g t^2 \implies t = \sqrt{\frac{2h}{g}} = \sqrt{\frac{40}{9.81}} = \sqrt{4.078}\\]
\\[x = v_0 \cdot t = 12 \times 2.02\\]

<span class="final-answer">t ≈ 2.02 s, x ≈ 24.23 m</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**11.**</span> \\(\theta = 50°\\), \\(T = 2.5 \, \text{s}\\) (level launch and landing).

\\[T = \frac{2 v_0 \sin\theta}{g} \implies v_0 = \frac{T g}{2 \sin\theta} = \frac{2.5 \times 9.81}{2 \times \sin 50°} = \frac{24.53}{1.532}\\]

so \\(v_0 \approx 16.01 \, \text{m/s}\\), and \\(v_0^2 \approx 256.3\\).

\\[R(50°) = \frac{v_0^2 \sin 100°}{g} = \frac{256.3 \times 0.9848}{9.81} \approx 25.73 \, \text{m}\\]
\\[R_{max}(45°) = \frac{v_0^2}{g} = \frac{256.3}{9.81} \approx 26.13 \, \text{m}\\]

<span class="final-answer">v₀ ≈ 16.01 m/s; the actual range (25.73 m) is only ≈ 0.4 m short of the maximum possible range at the same speed, since 50° is close to the optimal 45°</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**12.**</span> \\(v_0 = 80 \, \text{m/s}\\) (horizontal), \\(h = 150 \, \text{m}\\).

\\[t = \sqrt{\frac{2h}{g}} = \sqrt{\frac{300}{9.81}} = \sqrt{30.58}\\]
\\[x = v_0 \cdot t = 80 \times 5.53\\]
\\[v_y = g t = 9.81 \times 5.53 = 54.26 \, \text{m/s}\\]
\\[v_{impact} = \sqrt{v_0^2 + v_y^2} = \sqrt{80^2 + 54.26^2} = \sqrt{9344}\\]

<span class="final-answer">x ≈ 442.6 m, v_impact ≈ 96.7 m/s</span>
</div>

## ⭐⭐⭐⭐ Level 4 — Challenge
{: .section-heading }

<div class="solution-item" markdown="1">
<span class="sol-num">**13.**</span> \\(v_0 = 25 \, \text{m/s}\\), \\(\theta = 20°\\), cliff height \\(30 \, \text{m}\\) (landing point is 30 m below the launch point).

\\[v_{0x} = 25 \cos 20° = 23.49 \, \text{m/s}, \quad v_{0y} = 25 \sin 20° = 8.55 \, \text{m/s}\\]

Set \\(y(t) = -30 \, \text{m}\\):

\\[-30 = 8.55\,t - 4.905\,t^2 \implies 4.905\,t^2 - 8.55\,t - 30 = 0\\]
\\[t = \frac{8.55 + \sqrt{8.55^2 + 4 \times 4.905 \times 30}}{2 \times 4.905} = \frac{8.55 + \sqrt{661.7}}{9.81} = \frac{8.55 + 25.72}{9.81}\\]

(the negative root is discarded — time cannot be negative)

\\[x = v_{0x} \cdot t = 23.49 \times 3.49\\]

<span class="final-answer">T ≈ 3.49 s, R ≈ 82.1 m</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**14.**</span> \\(\theta = 55°\\), wall of height \\(6 \, \text{m}\\) at \\(x = 15 \, \text{m}\\).

Use the trajectory equation (position as a function of \\(x\\) instead of \\(t\\)):

\\[y(x) = x \tan\theta - \frac{g x^2}{2 v_0^2 \cos^2\theta}\\]

At the minimum speed, the ball just grazes the top of the wall, so set \\(y(15) = 6\\) and solve for \\(v_0\\):

\\[6 = 15 \tan 55° - \frac{9.81 \times 15^2}{2 v_0^2 \cos^2 55°} = 21.42 - \frac{2207.25}{0.658 \, v_0^2}\\]
\\[\frac{2207.25}{0.658 \, v_0^2} = 15.42 \implies v_0^2 = \frac{2207.25}{0.658 \times 15.42} \approx 217.5\\]

<span class="final-answer">v₀ ≈ 14.75 m/s</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**15.**</span> \\(v_0 = 25 \, \text{m/s}\\) for both; \\(\theta_A = 30°\\), \\(\theta_B = 60°\\).

Because \\(30°\\) and \\(60°\\) are complementary (\\(\sin(2 \times 30°) = \sin 60° = \sin(2 \times 60°) = \sin 120°\\)), both angles give the **same range**:

\\[R_A = R_B = \frac{v_0^2 \sin 60°}{g} = \frac{625 \times 0.866}{9.81} \approx 55.17 \, \text{m}\\]

But their flight times differ:

\\[T_A = \frac{2 v_0 \sin 30°}{g} = \frac{25}{9.81} \approx 2.55 \, \text{s}, \qquad T_B = \frac{2 v_0 \sin 60°}{g} = \frac{43.30}{9.81} \approx 4.41 \, \text{s}\\]

<span class="final-answer">Both land ≈ 55.17 m away; ball B (steeper launch) lands ≈ 1.86 s after ball A</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**16.**</span> Release height \\(y_0 = 1.5 \, \text{m}\\), \\(v_0 = 14 \, \text{m/s}\\), \\(\theta = 25°\\).

\\[v_{0x} = 14 \cos 25° = 12.69 \, \text{m/s}, \quad v_{0y} = 14 \sin 25° = 5.92 \, \text{m/s}\\]

Set \\(y(t) = y_0 + v_{0y} t - \tfrac{1}{2} g t^2 = 0\\):

\\[1.5 + 5.92\,t - 4.905\,t^2 = 0 \implies 4.905\,t^2 - 5.92\,t - 1.5 = 0\\]
\\[t = \frac{5.92 + \sqrt{5.92^2 + 4 \times 4.905 \times 1.5}}{2 \times 4.905} = \frac{5.92 + \sqrt{64.43}}{9.81} = \frac{5.92 + 8.03}{9.81}\\]
\\[x = v_{0x} \cdot t = 12.69 \times 1.42\\]

<span class="final-answer">T ≈ 1.42 s, R ≈ 18.0 m</span>
</div>

## ⭐⭐⭐⭐⭐ Level 5 — Expert
{: .section-heading }

<div class="solution-item" markdown="1">
<span class="sol-num">**17.**</span> \\(v_0 = 45 \, \text{m/s}\\), \\(\theta = 38°\\), fence: \\(15 \, \text{m}\\) high at \\(x = 60 \, \text{m}\\).

\\[v_{0x} = 45 \cos 38° = 35.46 \, \text{m/s} \implies t = \frac{x}{v_{0x}} = \frac{60}{35.46} \approx 1.69 \, \text{s}\\]
\\[y(t) = v_0 \sin\theta \cdot t - \tfrac{1}{2} g t^2 = 45 \sin 38° \times 1.69 - 4.905 \times 1.69^2 = 46.87 - 14.04\\]

\\(y \approx 32.8 \, \text{m}\\), which is well above the \\(15 \, \text{m}\\) fence.

<span class="final-answer">The ball clears the fence by ≈ 17.8 m</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**18.**</span> \\(v_0 = 30 \, \text{m/s}\\), \\(\theta = 50°\\), \\(y = 20 \, \text{m}\\).

\\[v_{0y} = 30 \sin 50° = 22.98 \, \text{m/s}\\]
\\[20 = 22.98\,t - 4.905\,t^2 \implies 4.905\,t^2 - 22.98\,t + 20 = 0\\]
\\[t = \frac{22.98 \pm \sqrt{22.98^2 - 4 \times 4.905 \times 20}}{2 \times 4.905} = \frac{22.98 \pm \sqrt{135.7}}{9.81} = \frac{22.98 \pm 11.65}{9.81}\\]

<span class="final-answer">t₁ ≈ 1.16 s (rising through 20 m) and t₂ ≈ 3.53 s (falling back through 20 m)</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**19.**</span> \\(H = 60 \, \text{m}\\), \\(v_0 = 40 \, \text{m/s}\\), \\(\theta = 30°\\).

\\[v_{0x} = 40 \cos 30° = 34.64 \, \text{m/s}, \quad v_{0y} = 40 \sin 30° = 20.00 \, \text{m/s}\\]

Set \\(y(t) = -60 \, \text{m}\\):

\\[-60 = 20\,t - 4.905\,t^2 \implies 4.905\,t^2 - 20\,t - 60 = 0\\]
\\[t = \frac{20 + \sqrt{400 + 4 \times 4.905 \times 60}}{2 \times 4.905} = \frac{20 + \sqrt{1577.2}}{9.81} = \frac{20 + 39.71}{9.81} \approx 6.09 \, \text{s}\\]

The impact speed can be found either from the velocity components or, faster, from energy conservation:

\\[v_{impact} = \sqrt{v_0^2 + 2 g H} = \sqrt{1600 + 2 \times 9.81 \times 60} = \sqrt{2777.2}\\]

<span class="final-answer">T ≈ 6.09 s, v_impact ≈ 52.7 m/s</span>
</div>

<div class="solution-item" markdown="1">
<span class="sol-num">**20.**</span> \\(v_0 = 25 \, \text{m/s}\\), \\(\theta = 50°\\), incline at \\(20°\\) from the launch point.

The incline surface is the line \\(y = x \tan 20°\\). The projectile lands where its trajectory meets this line:

\\[x \tan\theta - \frac{g x^2}{2 v_0^2 \cos^2\theta} = x \tan 20°\\]

Dividing through by \\(x\\) (the trivial solution \\(x = 0\\) is the launch point itself):

\\[x = \frac{2 v_0^2 \cos^2\theta \,(\tan\theta - \tan 20°)}{g} = \frac{2 \times 625 \times \cos^2 50° \times (\tan 50° - \tan 20°)}{9.81}\\]
\\[= \frac{2 \times 625 \times 0.4132 \times 0.8278}{9.81} = \frac{427.6}{9.81} \approx 43.6 \, \text{m}\\]

Finally, convert the horizontal distance into a distance measured **along the slope**:

\\[d = \frac{x}{\cos 20°} = \frac{43.6}{0.9397}\\]

<span class="final-answer">d ≈ 46.4 m along the incline</span>
</div>
