🎮 How to Use the Simulation

Step 1: Set the Radius

Use the "Radius r" input to set the size of the circular path in meters (0.5 to 5 m).

Step 2: Set the Angular Velocity

Use the "Angular velocity ω" input to set how fast the body sweeps angle, in radians per second. A higher ω means a shorter period T and a faster-spinning point.

Step 3: Choose a Direction

Pick counter-clockwise or clockwise rotation.

Step 4: Play

Click "Play" to start the motion. You'll see:

  • The blue body moving at constant speed along the dashed circular path
  • The green arrow — the velocity vector, always tangent to the circle
  • The red arrow — the centripetal acceleration vector, always pointing toward the center
  • Live readouts of elapsed time, angle θ, speed v, centripetal acceleration ac, period T, and frequency f

Step 5: Experiment

Try changing r and ω while the simulation is running (even mid-motion) and observe how:

  • The velocity vector length never changes direction relative to the radius — it's always perpendicular to it
  • The acceleration vector always points toward the center, no matter where the body is on the circle
  • Doubling ω quadruples the centripetal acceleration (ac = ω²r), while doubling r only doubles it
