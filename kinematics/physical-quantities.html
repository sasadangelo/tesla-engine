🎮 How to Use the Simulation

Step 1: Set the true values

Enter the true distance the car travels and the true time it takes. These are the "real" values that exist in nature — your instruments can only approximate them.

Step 2: Choose instrument resolutions

Select a ruler and a stopwatch from the dropdowns. A coarser instrument (larger resolution) produces a larger absolute error Δx = resolution / 2.

Step 3: Click "Measure!"

The simulation animates the car driving across the track. When it stops, the instruments display their readings — slightly different from the true values because of rounding to the nearest scale division. The stats panel shows the full error calculation.

Step 4: Experiment

Try switching to a coarser stopwatch and observe how the relative error on velocity jumps. Notice that a small timing error can dominate over a large distance error — this is why εᵣ(v) = εᵣ(Δx) + εᵣ(Δt) is the key formula.

📐 The Maths Behind the Simulation

Instrument reading

Each instrument rounds the true value to the nearest scale division (resolution). The absolute error is half that resolution:

Δx = resolution / 2    Δt = resolution / 2

Calculated velocity

v = x_measured / t_measured

Relative error propagation

εᵣ(v) = Δx / x_measured + Δt / t_measured

Absolute error on velocity

Δv = εᵣ(v) × v

Final result notation

v = (v_measured ± Δv) m/s

🧪 Suggested Experiments

Experiment 1 — Distance dominates

Setup: distance = 5 m, time = 8 s, ruler = 1 m, stopwatch = 0.01 s

Expected: Δx/x ≈ 10 %, Δt/t ≈ 0.06 % — the coarse ruler dominates the error budget.

Experiment 2 — Time dominates

Setup: distance = 200 m, time = 0.5 s, ruler = 0.1 m, stopwatch = 0.1 s

Expected: Δx/x ≈ 0.025 %, Δt/t ≈ 10 % — a very short time interval is hard to measure.

Experiment 3 — Best precision

Setup: distance = 100 m, time = 8 s, ruler = 0.01 m, stopwatch = 0.01 s

Expected: total εᵣ(v) < 0.1 % — both instruments are precise and the true value lies well inside the error bar.
