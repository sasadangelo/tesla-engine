🏁 Acceleration Race Application

In our simulation, three vehicles race along parallel tracks: 🏎️ Red Racer, 🚙 Blue Cruiser, and 🚕 Yellow Taxi. Each vehicle starts from rest (v₀ = 0) and accelerates uniformly at a constant acceleration that you can set.

1. Winner Prediction

The vehicle with the highest acceleration will always reach the finish line first, since they all start from the same position and speed (zero).

If a₁ > a₂ > a₃, then vehicle 1 wins

2. Finish Time Calculation

For a race distance D and starting from rest, each vehicle's finish time is derived from x = ½·a·t²:

t_finish = √(2D / a)

Example: For D = 500m and a = 5 m/s², the finish time is √(1000/5) = √200 ≈ 14.14 seconds.

3. Speed at Finish Line

Using v² = 2·a·D, the speed each vehicle reaches at the finish line is:

v_finish = √(2 · a · D)

Example: For a = 5 m/s² and D = 500m → v = √5000 ≈ 70.7 m/s.

🧪 Practical Examples

Example 1: Equal Accelerations

Setup: All three vehicles at a = 4 m/s²

Result: They all finish at the same time (tie)

Time: t = √(2×500/4) = √250 ≈ 15.81 seconds

Example 2: Double Acceleration

Setup: 🏎️ Red at 8 m/s², 🚙 Blue at 4 m/s²

Result: Red finishes in √2 ≈ 1.41× less time

Times:

  • 🏎️ Red: t = √(1000/8) = √125 ≈ 11.18 seconds
  • 🚙 Blue: t = √(1000/4) = √250 ≈ 15.81 seconds

Example 3: Gap Between Vehicles

Question: At t = 10s, how far apart are 🏎️ Red (a=5) and 🚕 Yellow (a=3)?

Answer: x₁ = ½×5×100 = 250m, x₂ = ½×3×100 = 150m → gap = 100m
